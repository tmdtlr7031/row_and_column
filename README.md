### mainAxisAlignment
- start : 시작
- end : 끝
- center : 중간
- spaceBetween : 위젯과 위젯의 여백이 균등하게 적용
- spaceEvenly : spaceBetween 효과지만 시작, 끝 부분에 위젯이 아닌 여백으로 시작
- spaceAround : spaceEvenly + 시작, 끝부분의 여백이 1/2

### crossAxisAlignment
- 기본값은 CrossAxisAlignment.center
- baseline : 글자 정렬 시 이용됨 (글자 밑부분 정렬)
- stretch : 최대한으로 늘린다
